# Bipedal Walker Reinforcement Learning Project

This project explores solving the [**OpenAI Gymnasium Bipedal Walker environment**](https://gymnasium.farama.org/environments/box2d/bipedal_walker/) using three deep reinforcement learning algorithms:

- Deep Deterministic Policy Gradient (DDPG)
- Twin Delayed Deep Deterministic Policy Gradient (TD3)
- Soft Actor-Critic (SAC)

Both **normal** and **hardcore** versions of the environment were investigated.
